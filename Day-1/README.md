# Data Science BootCamp

This repository contains beginner Python exercises for Day 1 of a Data Science BootCamp. The exercises are organized by difficulty level and demonstrate basic programming concepts such as prime number checking, Fibonacci sequence generation, and a simple interactive calculator.

## Project Structure

- `Day-1/Level-1/prime.py` - Checks whether a given number is prime.
- `Day-1/Level-2/fibonacci.py` - Generates the first `n` numbers of the Fibonacci sequence.
- `Day-1/Level-3/calculator.py` - Runs an interactive calculator with addition, subtraction, multiplication, and division.

## Requirements

- Python 3.x

## Running the Exercises

Open a terminal and run the desired script using Python. For example:

```bash
python "Day-1/Level-1/prime.py"
python "Day-1/Level-2/fibonacci.py"
python "Day-1/Level-3/calculator.py"
```

Follow the prompts displayed by each script.

## Notes

- `prime.py` prints `True` or `False` depending on whether the entered number is prime.
- `fibonacci.py` prints the first `n` Fibonacci numbers as a list.
- `calculator.py` provides a simple menu-driven calculator and handles division by zero.
